# 团子噜噜工作室

团子噜噜工作室的作品展示网站。

这是一个以作品为主的深棕色展厅式作品集，主要展示：

- 作品
- 定价入口
- 联系方式
- 作品前后对比
- 完整源文件查看入口

## 当前特点

- 使用原生 HTML、CSS 和 JavaScript，无后端、数据库和表单服务。
- 页面不依赖长页面滚动，作品、定价和联系状态在同一展厅内切换。
- 支持桌面、平板、手机竖屏和手机横屏。
- 采用流式布局、内容断点和安全区适配，避免横向溢出、内容被裁切或按钮太小。
- 作品素材暂时留空时，显示干净的待补充状态，不使用库存视频冒充作品。
- 联系方式支持复制账号、查看二维码和下载真实二维码图片。
- 页面支持键盘切换、触摸滑动和减少动效模式。

## 添加作品

作品数据集中写在 `index.html` 的 `works` 数组中。每件作品可以填写：

- 作品名称
- 作品类型
- 主画面图片或视频
- 前后对比图片
- 作品链接
- 简短说明
- 源文件状态

素材还没有准备好时，保留空值即可，页面会显示待补充状态。

## 添加定价图

定价入口目前保留为空，未来可以将图片放入：

```text
assets/pricing.png
```

然后在 `index.html` 中补充定价图片即可。

## 本地查看

直接双击 `index.html` 可以查看页面。若需要测试图片、二维码下载等功能，建议在项目目录启动一个本地静态服务器：

```bash
python -m http.server 8765
```

然后打开：

```text
http://127.0.0.1:8765/index.html
```

## 响应式适配

当前页面重点处理了以下情况：

- 手机窄屏：联系方式和按钮不会挤出页面。
- 手机横屏：内容不会因为屏幕高度较低而被裁掉。
- 平板：三栏展厅会根据可用宽度收缩，空间不足时自动调整。
- 刘海屏：使用安全区间距，避免 Logo、导航和底部内容贴边。
- 图片和二维码：不会撑破容器，保持原比例显示。
- 中文内容：使用 UTF-8 和中文系统字体回退，降低乱码风险。

## 更新网站

```bash

git commit -m "更新作品集和响应式适配"
git push
```

网站部署服务完成构建后即可看到更新。

## 说明

本项目不包含后端、数据库、CMS、用户表单或第三方库存视频。网站的重点是清楚展示真实作品，并让访客能够方便地查看作品、联系工作室和获取二维码。
